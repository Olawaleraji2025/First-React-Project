# Sneaker Product Page (React + Vite)

A React + Vite single-page e-commerce product UI featuring a sneaker gallery (thumbnails + modal viewer), product details (title/description/pricing + discount display), quantity controls, and an interactive cart experience (cart indicator + empty-cart modal) with a sidebar navigation.

## Features
- Product image gallery with thumbnail navigation
- Modal product viewer with next/previous controls
- Product details section (title, description, price + discount)
- Quantity selector with + / - controls
- “Add to cart” flow with cart indicator
- Responsive header + sidebar menu

## Tech Stack
- React
- Vite
- CSS 
- react-icons


## Notes
This project is a front-end UI demo (no backend/payment integration).
